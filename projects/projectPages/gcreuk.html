GCRE UK - Energy Storage System Project

A comprehensive energy storage solution utilizing advanced UCAP modules, CAN communication, and custom data decoding.

This project aimed to supply GCRE UK with a reliable, high-performance energy storage system powered by Skeleton UCAP modules. We configured these modules using the CAN protocol and built a data decoder with Arduino to monitor and analyze CAN data in real time. This solution provides scalable energy storage, efficient configuration, and precise data interpretation.

Project Goals

  • Design a scalable and efficient energy storage solution using UCAP modules.
  • Implement CAN communication for seamless configuration and data transmission.
  • Develop a custom data decoder to interpret and monitor CAN data.
  • Ensure reliability and adaptability for integration with future projects.

Technologies Used

CAN Protocol

Utilized for configuring UCAP modules and ensuring robust communication.

PCAN Adapter & PCAN Explorer

PCAN tools enabled efficient communication and configuration for the UCAP modules.

Arduino Nano & MCP2515 Module

Used to build the Node Receiver for decoding CAN data per GCRE UK's specific data format.

Node.js & JavaScript

Developed custom decoding scripts to ensure accurate data interpretation and integration.

Project Process

Step 1: UCAP Module Selection and Testing

We selected Skeleton UCAP modules for their energy density and reliability. Initial tests were conducted to verify compatibility with the CAN protocol and assess performance under load conditions.

Step 2: CAN Communication Setup

Using a PCAN adapter and PCAN Explorer, we configured the UCAP modules via CAN, allowing for smooth data transmission and remote configuration capabilities.

Step 3: Node Receiver Development

We designed a custom Node Receiver using an Arduino Nano with the MCP2515 CAN module. This receiver was programmed to decode incoming CAN data based on GCRE UK’s specifications, using a data format that ensured all necessary parameters were accurately captured.

Step 4: Testing & Validation

The Node Receiver was rigorously tested to verify data accuracy and ensure real-time decoding. The final setup allowed for robust monitoring and control of the UCAP modules in real-world scenarios.

Results

The GCRE UK project delivered a fully functional, reliable energy storage system capable of efficient configuration and data monitoring. By utilizing the CAN protocol, PCAN tools, and Arduino-based decoding, we met all project goals. The setup now allows GCRE UK to scale their energy storage capabilities with confidence.

Conclusion

This project showcases the potential of integrating advanced CAN communication with robust energy storage solutions. It provided a valuable learning experience in using tools like the PCAN adapter, CAN protocols, and Arduino-based decoding systems.
