TATA 340 PP 40

High-Voltage Energy Storage System with CAN-based Data Output

In the TATA 340 PP 40 project, we designed an advanced energy storage system, charging up to 340V and capable of delivering 40kJ of energy. The system features ultra capacitors (UCaps) with real-time voltage and current monitoring using Electrohms VH1K0T01 voltage sensors and HE100T01 Hall Effect-based current sensors. The CAN protocol was used to transmit key data, enabling external systems to monitor and interact with the storage system in real time.

Project Goals

  • Develop a high-voltage, high-capacity energy storage system with accurate monitoring.
  • Implement CAN protocol for reliable data transmission to external systems.
  • Ensure stable communication with external control systems through standardized data format.

Technologies Used

UCaps

3V-rated Ultra Capacitors stacked to achieve high voltage, with a total capacity of 340V and 40kJ.

CAN Protocol

Used to transmit system data to the outside world, allowing real-time external monitoring and interaction.

VH1K0T01 Voltage Sensor

Electrohms high-precision voltage sensor, ensuring accurate voltage readings in real time.

HE100T01 Current Sensor

Hall Effect-based sensor for precise current measurement, integral for system monitoring.

Project Process

System Design and Component Selection

The design involved stacking UCaps to reach the required voltage and energy capacity. Each component, including voltage and current sensors, was selected for reliability and precision, essential for high-voltage applications.

Master and Slave Board Development

Custom boards were developed to monitor and control the system. The master board collects sensor data and transmits it externally via CAN protocol, enabling real-time monitoring and control from outside systems.

CAN Protocol Integration and Testing

CAN protocol was implemented on the master board to facilitate reliable data output to external systems. Testing ensured accurate data transmission for real-time monitoring and control.

Project Results

Successfully implemented a high-capacity energy storage system with a robust CAN-based data transmission setup. This solution enables real-time external monitoring, providing stable and reliable data output.

Conclusion

This project demonstrates the effectiveness of using UCaps for high-voltage energy storage with accurate monitoring and CAN protocol for seamless data transfer to external systems.
